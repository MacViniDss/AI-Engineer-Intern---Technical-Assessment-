# YouTube Location Video Finder

## Ideia's Concept:
A Python backend application that fetches YouTube videos based on a given geographic location.

Users can input a location (city, ZIP code, coordinates, or landmark), and the system retrieves nearby videos using the YouTube Data API v3.

## Features
- Location-based video search
- CLI and REST API interface
- Export results to:
  - JSON
  - Markdown
  - PDF
- Modular and scalable architecture

## Tech Stack
- Python
- Flask
- YouTube Data API v3
- Geopy (geocoding)
- ReportLab (PDF generation)
